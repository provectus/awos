# AWOS - Product Summary

## Vision

To transform chaotic AI prompting into systematic spec-driven development, enabling developers and teams to generate high-quality code by providing AI agents with the structured context they need to succeed on the first try.

## Target Audience

- Solo developers using AI coding assistants
- Engineering teams seeking structured AI workflows
- Technical founders building MVPs with AI
- Business users involved in product and roadmap planning

## Core Features

- **Sequential Workflow Commands:** 7-step guided path from vision to code
- **Document-Driven State:** Markdown-based context, fully idempotent and resumable
- **Specialized Subagents:** Domain experts for React, Python, Kotlin
- **Extensibility:** Customizable commands and hooks for team workflows
- **Template System:** Pre-built templates for all specification types
